SOLDIER INFORMATION TRACKING SYSTEM

   This application is designed to keep track all information of a soldier right from his birth, the information can be maintained and updated in an centralized location and it can access globally.

STEGANOGRAPHY USING IMAGES AND CRYPTOGRPHY TOOLS

   The advantages of Steganography is that it can be used to secretly transmit messages without the fact of the transmission being discovered, Used the LSB(Least Significant Bit) Algorithm in this project for developing the application which is faster and reliable and compression ratio moderate compared to other algorithms.
